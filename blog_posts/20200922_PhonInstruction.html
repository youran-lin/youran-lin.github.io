<<  2020‧09‧22 |【Phon】Beginner Instructions of Phon

I developed step-by-step instructions for beginners to use Phon, a program that supports the building of textual and phonological data corpora. This instruction intends to show students, clinicians, and researchers some basics about using Phon to transcribe a recorded media file.

The wordlist is based on GFTA-3, a commonly used clinical assessment tool for child speech sound disorders.

The document will show learners how to:

  • import a template (wordlist) and a media file
  • segment the file according to the wordlist
  • transcribe the words/utterances ("records")
  • syllabify and align the target and transcription
  • print a Phon file to a PDF
  • segment and transcribe without template
